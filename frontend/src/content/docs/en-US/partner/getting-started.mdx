# Getting started

Phaeno Portal gives Partner organizations a tenant-isolated workspace for reagent orders, data-assembly requests, released outputs, curated data, and organization access.

## Select the Partner organization

Use the organization switcher when your account has multiple memberships. Prices, offerings, requests, files, and permissions always come from the selected organization. A Partner cannot see another Partner’s negotiated prices or work.

## Know your access level

Partner organization administrators can create and place reagent orders, create and submit data-assembly requests, accept applicable commercial changes, decide substitutions, request cancellations, and manage members. Other active Partner members have read, progress-tracking, and eligible output-download access.

## Find your work

- **Reagent orders** contains the organization’s available products, drafts, placed orders, substitutions, and shipments.
- **Data assembly** contains input revisions, quotes, processing status, and released outputs.
- **Data Library** contains Phaeno-curated packages explicitly assigned to the Partner.
- **Documentation** opens the Partner guides.

## Protect submitted data

The initial workflows are not designed to collect protected health information. Do not place patient identifiers, credentials, or unnecessary confidential information in project names, notes, file names, manifests, or uploads.

Use a Partner-controlled project reference and follow the metadata and file requirements shown by the selected assembly profile.

## Get help

Provide the portal order, request, shipment, or project reference when contacting Phaeno. Never include passwords, authentication tokens, or sensitive file contents in a support message.
