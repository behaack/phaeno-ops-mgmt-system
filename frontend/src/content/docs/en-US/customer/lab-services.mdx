# Request laboratory services

Customer organization administrators use Lab services to prepare a job, describe its samples, receive a job-specific quote, and authorize work.

## Create a draft

1. Open **Lab services** and start a request.
2. Choose an available analysis.
3. Enter the job reference and the required information for every sample.
4. Review the analysis instructions and expected outputs.
5. Save the draft and return later if more information is needed.

Drafts remain editable. Other members can see organization work but cannot submit or commercially approve it unless they are an organization administrator.

## Submit for review

Submission validates the current request and freezes an input revision. Phaeno can accept the information for quoting, request specific corrections, or reject work that cannot be performed.

When corrections are requested, create and submit a new revision. The prior revision remains in the job history; it is not silently overwritten.

## Review and accept a quote

Laboratory services are priced per job. Phaeno issues an itemized, immutable quote for the accepted scope. The quote is available only after its QuickBooks Online estimate has synchronized successfully.

Before accepting, confirm the sample list, analysis, price, currency, and expiration date. Only a Customer organization administrator can accept the current unexpired quote.

If the scope changes after acceptance, Phaeno issues a separate change quote. Added work does not begin until the applicable change is accepted.

## Ship and track samples

Follow the shipping instructions shown for the job. The portal tracks receipt, accessioning, processing, external sequencing, data processing, scientific review, holds, and readiness for release. A job can remain in progress while different samples move through laboratory work.

If Phaeno needs replacement material or clarification, the job shows a Customer-safe action and the current schedule health. Internal scientific investigation details are not shown.

**Ready for release** means Phaeno's scientific review is complete. It does not mean a file is downloadable. Commercial publication, file scanning, and any payment or credit gate must still pass before a result appears under Files and results.

## Cancel work

A draft can be discarded before submission. Before Phaeno accepts work, an administrator can cancel eligible scope. After acceptance, submit a cancellation request. Phaeno records whether the request is approved, partially approved, or declined, and any financial adjustment remains linked through QuickBooks Online.
