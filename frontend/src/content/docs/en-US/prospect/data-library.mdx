# Use the Data Library

The Data Library is a read-only catalog of Phaeno-owned curated data package versions explicitly granted to your Prospect organization. A grant applies only to the named package version; access to one version does not automatically include earlier or later versions.

## Review a granted package

1. Open **Data Library** in the current Prospect organization.
2. Select a package to review its summary and granted version.
3. Review the manifest, file names, sizes, and checksums before downloading.
4. Download an individual file or the package archive when needed.
5. Preserve the package version and checksum with your internal record of the download.

The package summary and manifest describe the curated release. They do not create a service request, place an order, or authorize use beyond the terms agreed with Phaeno.

## Understand access boundaries

- Phaeno must explicitly grant each package version to the organization.
- A newly published version does not replace or extend an existing grant.
- Members of another organization cannot use your organization’s grant.
- The portal does not allow a Prospect to upload, edit, publish, or grant packages.
- Download activity is recorded for governance and support.

If a package you expected is absent, provide Phaeno with the organization, package name, and expected version. Do not request access by sending copied data or another organization’s package details.

## Verify a download

Use the displayed checksum to verify that a downloaded file or archive is complete and unchanged. If a checksum does not match, delete the incomplete copy and retry once. If the second download also fails, stop and contact Phaeno with the package version, file name, time of the attempt, and the safe error message shown by the portal.
