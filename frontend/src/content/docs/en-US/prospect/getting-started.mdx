# Getting started

Phaeno Portal gives a Prospect organization read-only access to Phaeno-owned curated data packages that Phaeno has explicitly granted to that organization. Prospect access does not include laboratory requests, reagent orders, data-assembly requests, or other commercial ordering.

## Confirm the current organization

The user menu shows the organization active in your current session. That organization controls every record and action you can see, and information never carries between organizations.

Self-service switching between multiple external organization memberships is not currently available. If the wrong organization is active, or your account belongs to more than one external organization, contact Phaeno before continuing.

## Know your access level

Active Prospect members can review and download the exact curated package versions granted to their organization. Organization administrators can also review organization download history and may be asked to respond to a package-governance notice.

The current **User management** screen is a session-only preview. Changes made there do not update durable invitations or memberships; contact Phaeno for access changes.

## Find your work

- **Dashboard** provides a starting point for the current organization.
- **Data Library** contains curated package versions explicitly granted to the Prospect.
- For organization administrators, **User management** is a session-only preview and does not persist changes.
- **Documentation** opens these Prospect guides.

Prospect organizations do not receive order or service workspaces. Phaeno alone can change the organization relationship after business approval.

## Get help safely

When contacting Phaeno, provide the organization name, package name and version, and a description of the step you were performing. Do not send passwords, sign-in codes, invitation links, authentication tokens, or downloaded data in a support message.
