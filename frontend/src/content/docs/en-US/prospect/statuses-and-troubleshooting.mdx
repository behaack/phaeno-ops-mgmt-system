# Statuses and troubleshooting

Use the visible organization, package version, grant state, and safe portal message to understand a problem before contacting Phaeno.

| What you see | Meaning | What to do |
| --- | --- | --- |
| No packages | No active package version is currently granted to the organization, or the expected grant is unavailable. | Confirm the organization shown in the user menu, then ask Phaeno to review the exact package version. |
| Package or version missing | Grants are version-specific and do not move automatically to a new release. | Provide Phaeno with the expected package and version. |
| Access revoked | Phaeno ended the organization’s future access to that version. | Stop retrying and follow any governance instruction from Phaeno. |
| Version quarantined | Future access is blocked while Phaeno reviews the release. | Do not use another link or another organization’s copy. Wait for Phaeno’s instruction. |
| Package retired | The package is no longer available for new portal use. | Ask Phaeno whether an approved replacement exists. |
| Download failed | The file or archive did not complete successfully. | Retry once. If it fails again, report the file name, package version, time, and safe error message. |
| Checksum mismatch | The downloaded content is incomplete or changed. | Delete that copy, retry once, and contact Phaeno if the second checksum also fails. |
| Wrong organization shown | The current session is scoped to a different membership. | Stop before opening or downloading data and contact Phaeno. |
| Order action missing | Prospect organizations cannot place laboratory, reagent, or data-assembly orders. | Contact Phaeno if the business relationship should be reviewed. |

## Report a problem safely

Include the organization name, package name and version, file name when applicable, time of the event, and the portal’s safe error message. Do not attach downloaded data or send passwords, sign-in codes, invitation tokens, access tokens, browser storage, or another organization’s information.
